TERMOS DE USO DA API DO SISTEMA WEB DE GERENCIAMENTO DE EVENTOS (SWGE) - LATINOWARE

Este documento explica e regula como o SWGE-LATINOWARE disponibiliza conteúdos por meio de Interface de Programação de Aplicação (API), e seu funcionamento, para gerenciar e esclarecer que a integração entre plataformas e usuário e parceiros estejam em conformidade com a Lei Geral de Proteção de Dados Pessoais – LGPD, Lei 13.709/2018.

Glossário

  • API, termo para se referir a Interface de Programação de Aplicação fornecida pelo Sistema Web de Gerenciamento de Eventos;
  • SWGE-LATINOWARE, termo para se referir ao Sistema Web de Gerenciamento de Eventos;
  • GPLv3, termo que se refere a versão 3 da Licença Pública Geral GNU (2007).

  1. Objeto
  1.  Uma Application Programming Interface (API) ou em tradução para o português "Interface de Programação de Aplicação" facilita a troca de informações entre sistemas e linguagens de programação diferentes, por meio de instruções e protocolos.
  2.  Ela funciona de maneira que, pode ser explicada em termos de cliente e servidor, sendo assim, quem envia uma solicitação dos dados é chamada de cliente e quem envia os dados é chamado de servidor.

  1. Licenciamento
  1.  O presente Termo define as regras para uso da API do SWGE-LATINOWARE. Sujeito as regras definidas pela documentação do sistema e da lei de número 13.709/2018 Lei Geral de Proteção de Dados Pessoais (LGPD).
  2.  O usuário desenvolvedor poderá utilizar API, de acordo com a licença GPLv3, estabelecida no desenvolvimento do SWGE-LATINOWARE.
  3.  O usuário desenvolvedor poderá anular uso da presente licença simplesmente descontinuando a utilização da API, bem como o SWGE-LATINOWARE pode anular uso pelo usuário caso verifique-se a violação dos Termos de Uso do WGE-LATINOWARE ou de sua Política de Privacidade.
  4.  Em caso de uso abusivo e indevido da API, o SWGE-LATINOWARE pode bloquear o acesso aos dados compartilhados.
  5.   Em caso de descontinuidade de um evento, o SWGE-LATINOWARE pode anular a utilização da API.

  1. Obrigações do Usuário
  1.  Não é permitido o uso da API de forma que possa violar qualquer Lei ou direito de terceiro, de maneira fraudulenta ou não autorizada.
  2.  Não é permitida a transmissão de qualquer tipo de código malicioso que possa danificar, interferir ou bloquear dados ou o sistema SWGE-LATINOWARE.
  3.  Não é permitido o uso dos dados recebidos por meio da API, que possibilite de qualquer maneira:
  1. Violação dos direitos do titular garantido pela lei n° 13.709/2018 Lei Geral de Proteção de Dados Pessoais (LGPD);
  2. Descumprimento dos direitos assegurados pela licença GPLv3.

  1. Privacidade e Proteção dos Dados Pessoais
  1.  O usuário fica ciente que não possui nenhum direito independente sobre dados pessoais tratados.
  2.  O tratamento de dados pessoais pode ser realizado quando com o devido consentimento do titular dos dados pessoais, nos termos da Lei nº 13.709/2018.
  3.  Evitar praticar qualquer atividade que possibilite riscos à privacidade dos usuários titulares dos dados pessoais, zelando pela manutenção e segurança do local de armazenamento dos dados tratados.
  4.  O usuário deve notificar o suporte do SWGE-LATINOWARE em caso de falha na Segurança de Informação que envolva os dados pessoais dos titulares consumidos por meio desta API.
  1. Caso o usuário não notifique o suporte do SWGE-LATINOWARE, ou demore um tempo longo até que possam ser reparados possíveis danos, o mesmo fica responsável pelos danos causados aos terceiros, cabendo direito ao SWGE-LATINOWARE de regresso em caso de condenação judicial ou extrajudicial, previsto na Lei N° 10.406 (Código Civil);

  1. Dos Dados Pessoais Transmitidos
  1.  Em conformidade com o disposto pela LGPD, o SWGE-LATINOWARE é a Operadora, enquanto que o seu usuário e/ou parceiro exerce a função de Controlador dos dados.
  1. Quando pelo controlador, referem à pessoa natural ou jurídica, de direito público ou privado, a quem competem as decisões referentes ao tratamento de dados pessoais;
  2.     Quando operadores, remete a decisões referentes à pessoa natural ou jurídica, de direito público ou privado, que realiza o tratamento de dados pessoais em nome do controlador;
  1.  Sobre a API:
  1. A transmissão dos dados pessoais através da API é feita de forma segura, por meio de um canal criptografado, utilizando o protocolo HTTPS – Hyper Text Transfer Protocol Secure, não sendo aceitas requisições via protocolo HTTP – Hyper Text Transfer Protocol;
  2. O formato de comunicação dos dados é JSON – JavaScript Object Notation;
  3. A API disponibiliza duas formas de acessar os dados:
  1. Disponibilizamos publicamente as edições de um evento, caravanas, atividades, trilhas e a quantificação de participantes categorizados por cidade, estado, país e gênero.
  1. A finalidade do uso dos dados caracterizados como pessoais e sensíveis compartilhados através da API pública está descrita em nossa Política de Privacidade .
  1. Disponibilizamos de forma privada os demais dados não citados no item acima, aos quais são limitados baseados nas permissões que o usuário autenticado possui, mais informações sobre os tipos de autenticação podem ser vistas em nosso Termo de Uso .
  1. O token de acesso é gerado ao efetuar login e é passado subsequentemente para cada requisição realizada;

  1. Limitação de Responsabilidade
  1. O SWGE-LATINOWARE não é responsável, bem como não possui obrigação, em controlar as ações dos usuários desta API.
  2.  O SWGE-LATINOWARE não poderá ser responsabilizado pelos atos de seus usuários, inclusive desenvolvedores, sendo eles os cabíveis de responder por possíveis reclamações ou queixas judiciais e de envolvidos.

  1. Comunicação
  1.      Em caso de qualquer dúvida, sugestão ou problema com a utilização da API, o usuário poderá entrar em contato com o suporte, por meio do e-mail latinoware@pti.org.br .

  1. Disposições Gerais
  1.  Os termos e documentação técnica contidos nesta Política do Desenvolvedor têm duração indefinida e poderão ser modificados a qualquer momento para adequações e inserções, visando ao aprimoramento dos serviços disponibilizados.
